## Notas de aula [pdf](/01-Introdução.pdf)

1. **Computadores digitais**
2. Composição de um computador digital: **hardware** (fisicamente tangível) e **software** (fisicamente intangível)
3. Hardware: **entrada**, **saída**, **entrada/saída**, **unidade de processamento**
4. Software: sistema operacional, editor de texto, navegador, **assistentes inteligentes**, **jogos eletrônicos**, etc
5. **Hardware executa instruções binárias**: linguagem binária ou de máquina com 2 caracteres zero ou um
6. **Humanos** criaram **linguagens de programação** que são traduzidas para linguagem binária
7. **Tradução de linguagem de programação** para linguagem de máquina: **compilador** ou **interpretador**
8. **Compilador** produz (cria) arquivo(s) com linguagens de máquina
9. **Tradutor** lê código-fonte, traduz para linguagem de máquina e executa as instruções
